# MapleStory Style Transfer Prompts

## Core Style Description

Always include these descriptors for authentic MapleStory feel:

```
MapleStory 2D side-scrolling game style, pixel art aesthetic, vibrant colors, cute chibi proportions, hand-painted background layers, warm nostalgic game art
```

## Prompt Templates by Category

### Background / Landscape

```
Transform this image into a MapleStory 2D side-scrolling game background. Pixel art style with hand-painted layers, vibrant saturated colors, parallax-ready composition. {scene_context}. Warm nostalgic RPG atmosphere, cute and inviting game world aesthetic, detailed environmental storytelling.
```

Scene context examples:
- Henesys: "peaceful mushroom village, sunny afternoon, green rolling hills"
- Ellinia: "enchanted treehouse forest, mystical green glow, ancient trees with rope bridges"
- Kerning City: "urban neon-lit streets, subway platforms, rainy night atmosphere"
- Ludibrium: "colorful toy castle, candy blocks, whimsical floating platforms"
- Aquarium: "underwater coral reef, bubbles, bioluminescent sea creatures"

### Character / Portrait

```
Transform this person/character into a MapleStory chibi sprite style. 2D pixel art, large expressive eyes, small body with big head proportions (3-4 head ratio), cute simplified features, clean outlines, vibrant anime-game coloring. Full body with {pose}. Transparent background.
```

Pose options: "front-facing idle pose", "walking animation sprite sheet (front, back, left, right)", "action pose"

### Object / Item

```
Transform this object into a MapleStory in-game item icon. 2D pixel art style, clean outlines, vibrant colors, slight cel-shading, game UI icon aesthetic, small detailed sprite. Transparent background.
```

### Scene with Characters

```
Transform this scene into a MapleStory in-game screenshot. 2D side-scrolling perspective, pixel art characters with chibi proportions, hand-painted background layers, game UI elements optional, warm nostalgic RPG atmosphere. {scene_context}.
```

## Style Modifiers

| Modifier | Add to prompt |
|----------|---------------|
| Night | "nighttime palette, moonlit, deep blue-purple sky, warm window lights" |
| Rain | "rainy atmosphere, puddle reflections, grey overcast, cozy indoor warmth" |
| Snow | "winter snow, frost-covered trees, soft white overlay, warm village lights" |
| Autumn | "fall colors, orange-red leaves, golden hour warmth, harvest atmosphere" |
| Fantasy | "magical particles, glowing effects, enchanted atmosphere, ethereal light" |
| Urban | "city buildings, neon signs, subway, modern structures with pixel art style" |
| Retro | "nostalgic old-school pixel art, limited palette, classic MapleStory beta feel" |

## Tips for Better Results

1. **Include ingredient images**: Upload 1-3 MapleStory reference screenshots as ingredients for style consistency
2. **Be specific about composition**: "side-scrolling perspective", "parallax layers", "platform game layout"
3. **Reference MapleStory directly**: Include "MapleStory" and specific map names for Nano Banana Pro to understand the target style
4. **Aspect ratio matters**: 16:9 for backgrounds/landscapes, 1:1 or 9:16 for characters
5. **Transparent backgrounds**: Add "transparent background, PNG" for sprites and items
